# ShadowingChallenge
ShadowingChallenge is an web app to check one's pronunciation while shadowing or dictation with Web Speech API.

# Live Demo
[https://teonsen.github.io/ShadowingChallenge/](https://teonsen.github.io/ShadowingChallenge/)
